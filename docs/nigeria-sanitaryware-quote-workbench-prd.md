# 尼日利亚卫浴陶瓷报价工作台 PRD

## 1. 产品定位

本系统是面向尼日利亚卫浴陶瓷工厂的多人在线报价工作台。它不只是生成报价单，而是在正式接单前完成产品组合、配件路径、包装版本、装柜、物流、汇率、毛利、审批和历史追溯的统一核价。

第一版以 `FOB / CFR / CIF` 自动报价为核心。`DDP` 不进入自动报价，仅作为手动项目报价并要求主管审批。

## 2. MVP 边界

### 第一版必须交付

- 多人在线报价、用户角色、基础审批和报价状态流转。
- 产品库、包装版本、BOM 版本、供应商报价库。
- 中国配件四种路径：到尼厂组装后出货、到尼厂不组装随货出、中国直发客户、客户单独报价。
- 报价行级贸易条款：出货地、出货港、目的港、贸易条款、是否进入套装价、是否客户可见、是否单独报价、是否参与整柜装载。
- FOB/CFR/CIF 自动报价，含海运、保险、物流摊销和风险预留。
- 汇率快照、产品/BOM/包装/供应商/物流参数快照、报价版本。
- 客户版 PDF 与内部核价 Excel。
- 基础风险检查和基础装柜测算：CBM、毛重、柜数、装载率、物流摊销。

### 第一版简化

- 柔性装柜先做规则推荐，不做复杂组合优化。
- 嵌套装箱先支持手动维护嵌套后 CBM。
- 物流、汇率、CRM/ERP 接口先预留字段，第一版手动维护。
- 高级报表放第二阶段。

## 3. 关键业务对象

### 报价对象

- 单一产品。
- 成套产品。
- 主产品加可选配件。
- 配件单独报价。
- 陶瓷与中国配件分项报价。

### 供应来源

- 尼日利亚自产。
- 尼日利亚本地采购。
- 中国供应商采购。
- 手动确认成本。

### 成本口径

- 尼厂出厂成本。
- 尼厂装柜成本。
- 中国 EXW。
- 送仓/进仓价。
- FOB 中国港。
- 到尼厂入库成本。
- 手动确认成本。

已含物流的成本口径不得重复加物流。尤其是中国配件如果已经是到尼厂入库成本，就不能再叠加中国到尼日利亚中转物流。

## 4. 报价状态流转

```text
草稿 Draft
→ 待核价 Cost Reviewing
→ 待审批 Pending Approval
→ 已批准 Approved
→ 已外发 Issued
→ 客户接受 Accepted
→ 客户拒绝 Rejected
→ 已转订单 Converted
→ 已过期 Expired
→ 已作废 Cancelled
```

规则：

- 草稿可以自由修改。
- 提交审批后锁定关键成本字段。
- 已批准后只能外发，不能直接改价。
- 已外发后修改必须生成新版本。
- 已转订单后报价版本永久锁定。
- 已作废报价不能继续外发，只能复制为新草稿。
- 回滚只能生成新版本，不能覆盖旧版本。

## 5. 角色权限

| 角色 | 主要权限 |
| --- | --- |
| 业务员 | 新建报价、复制报价、选择产品、生成客户版 PDF |
| 采购 | 维护供应商、供应商报价、采购附件 |
| 物流/单证 | 维护港口、航线、柜型、运价、保险、港杂费 |
| 财务 | 维护汇率、付款条款、佣金、汇率风险规则 |
| 产品/工程 | 维护产品规格、包装、BOM、嵌套规则 |
| 主管 | 审批低毛利、DDP、手动成本、过期报价使用、高金额报价 |
| 管理员 | 用户权限、基础参数、审计日志 |

客户版 PDF 只允许业务员、主管、管理员生成。已审批报价撤回只允许主管和管理员操作。

## 6. 有效期规则

系统分别维护以下有效期：

- 产品成本有效期。
- 供应商报价有效期。
- 海运费有效期。
- 保险费有效期。
- 汇率有效期。
- 客户报价有效期。

客户报价未过期但运价、供应商报价、保险模板或汇率已失效时，系统必须提示复核。客户版默认显示：

`This quotation is valid for X days. Freight is subject to final confirmation at booking date.`

## 7. 风险检查分级

### 阻断类

- 成本缺失。
- 低于底线毛利。
- CIF 缺保险。
- DDP 未审批。
- 手动成本未审批。
- 客户版泄露内部字段。
- 已含物流成本重复加物流。

### 审批类

- 使用过期供应商报价。
- 使用过期物流费率。
- 毛利低于目标但高于底线。
- 手动修改装柜方案。
- 手动修改销售价。
- 使用自定义汇率。
- 高金额报价。

### 提醒类

- 装载率低。
- 高于客户历史价较多。
- 汇率接近阈值。
- 客户报价有效期长于成本有效期。
- 包装资料不完整但不影响当前报价。

## 8. 输出模板

### 客户版 PDF

包含：封面、公司信息、客户信息、报价编号、报价日期、有效期、贸易条款、目的港、付款方式、交期、产品图片、型号、描述、规格、包装、数量阶梯、单价、总价、备注、免责声明、签章区域。

禁止出现：供应商名称、采购价、BOM 成本、物流成本拆分、毛利、加价率、审批意见、内部备注、历史底价、风险检查。

展示方式：

- 套装总价。
- 分项报价。
- 主产品加可选配件。

### 内部核价 Excel

包含：报价总览、客户信息、贸易条款、数量阶梯、产品/BOM 明细、包装明细、供应商报价来源、成本口径、物流路径、装柜测算、海运/保险、汇率快照、毛利分析、历史报价对比、风险检查、审批记录、操作日志摘要。

## 9. 验收场景

- 单品马桶，尼日利亚自产，FOB 尼日利亚港。
- 马桶加中国盖板和水件，到尼厂组装后同箱出货。
- 马桶加中国配件，到尼厂不组装，随柜分包装出货。
- 尼日利亚陶瓷加中国配件直发客户，客户版分项报价。
- 台盆单品加可选下水件，客户选择是否含配件。
- EXW 中国供应商未填物流时阻断外发。
- 到尼厂入库成本又加中国段物流时阻断外发。
- CIF 缺保险时阻断外发。
- 已外发报价修改后必须生成新版本。
