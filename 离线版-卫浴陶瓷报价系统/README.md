# 卫浴陶瓷离线报价系统

这是离线版专用目录，只保留浏览器运行所需的静态文件：

- `index.html`：系统入口
- `styles.css`：页面样式
- `app.js`：报价、公式、导出逻辑
- `cloud.js`：IndexedDB 本机保存、搜索、版本、备份/恢复
- `卫浴陶瓷报价模板.xlsx`：给同事直接使用的无宏 Excel 报价模板
- `同事使用说明.md`：给业务和跟单同事的上手说明
- `新同事快速上手卡.md`：一页版操作检查卡

## 使用方式

直接用最新版 Edge 或 Chrome 打开 `index.html` 即可使用。报价数据保存在本机浏览器 IndexedDB 中，可在系统内使用“导出备份 / 导入备份”做 JSON 备份和恢复。给同事交接时，建议同时发 `同事使用说明.md` 和 `新同事快速上手卡.md`。

如果同事只需要填数据、核价和导出报价单，也可以直接使用 `卫浴陶瓷报价模板.xlsx`。该模板不含宏，打开后先填 `Input` 工作表，客户报价看 `CustomerQuote`，内部成本复核看 `InternalCost`。

## 离线版范围

本目录不包含旧版 Node 服务、MySQL、部署脚本、云函数、`node_modules` 或账号登录相关依赖。界面也已移除销售、采购、生产、财务、基础资料等未启用的 ERP 占位模块，只保留报价、成本参数和离线数据管理。

## 报价口径

- 客户版报价只显示销售价、数量阶梯、贸易条款、装运港/目的港、币种、有效期、MOQ 和交期。
- 内部核价按目标毛利反推销售价，并把佣金、售后和汇率风险按销售额预留，避免只按成本加点导致毛利被低估。
- 数量阶梯价会按每个数量档重新摊销物流和柜数，适合比较小单、整柜和多柜报价差异。
