# 新同事快速上手卡

## 第一次使用

1. 用 Edge 或 Chrome 打开 `index.html`。
2. 进入“系统管理 / 离线数据”，检查公司英文名、付款条款、报价备注。
3. 以后每天结束前点击“导出备份”，保存 JSON 备份文件。

## 做一张报价

1. 进入“报价工作台”。
2. 选产品模板：分体马桶、连体马桶、挂墙马桶、台上盆、台下盆、立柱盆/半柱盆。
3. 填客户、型号、数量、贸易条款、装运港、目标市场。
4. 马桶补：冲水方式、排污方式、坑距/墙排、是否含盖板。
5. 台盆补：安装方式、龙头孔、溢水孔、尺寸、净重。
6. 点“保存到本机”。

## 快速报价还是完整模式

- 只是先回客户一个区间：用“极速报价”。
- 准备正式发客户：切到“完整模式”。
- 需要主管看成本和利润：导出“内部Excel”。
- 准备发客户：导出“客户版PDF/打印”。

## 发客户前必须检查

- 客户名称和项目名称是否正确。
- 型号、尺寸、净重、包装是否正确。
- 贸易条款、装运港、目的港、币种、有效期是否清楚。
- MOQ 和交期是否能做到。
- 马桶是否漏了盖板、铰链、冲水和坑距。
- 台盆是否漏了龙头孔、溢水孔、下水/安装件。
- 客户版报价单里没有成本、利润和供应商信息。

## 价格异常时先看这里

- 偏高：检查数量是否太小、物流是否按 DDP/CIF、包装尺寸是否过大、认证/测试是否多选。
- 偏低：检查是否漏了盖板、下水件、认证、包装、目的港费用、汇率风险和售后风险。
- 阶梯价变化大：通常是物流和固定费用被不同数量重新摊销，不是系统错误。

## 备份提醒

离线版数据保存在当前电脑的当前浏览器里。换电脑、清理浏览器、重装系统前必须先“导出备份”。
